Collaborative Research: Framework: Machine Learning Materials Innovation Infrastructure

CSSI is a National Science Foundation (NSF) funded project with the goal of building materials centered cyberinfrastructure. The infrastructure built is two-fold: machine learning and data hosting. Efforts for materials machine learning are headed by the development team of the MAterials Simulation Toolkit for Machine Learning (MAST-ML). Data and model hosting, along with useful meta-data, is performed by the Foundry team.

MAST-ML provides a cohesive environment for training and testing standard machine learning models along with useful, automatically generated visuals. MAST-ML differs from other machine learning tools by taking a specialized approach to the material science research domain. Included research capacities include but are not limited to automatic elemental feature generation, cross-validation splits according to chemistry, uncertainty analysis, feature selection, etc.

The Foundry machine learning and data platform provides the materials science community intuitive and standardized methods of hosting models and data. The platform enables material scientists to easily verify and build upon past research efforts. Hosted data contains metadata guiding researchers to appropriate data fields with descriptions and hosted models are unambiguously liked to their corresponding training data set.
